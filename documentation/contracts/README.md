# Rebuild Contracts

These contracts are generated from live code and are intended to guide
module-by-module reconstruction work.

## Files

- `documentation/contracts/api-contract-map.json`
- `documentation/contracts/api-contract-map.md`
- `documentation/contracts/frontend-state-map.json`
- `documentation/contracts/frontend-state-map.md`

## Regenerate

Run:

```bash
python scripts/audit/generate_rebuild_contracts.py
```
