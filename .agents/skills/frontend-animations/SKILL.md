# Frontend Animations Skill for Kimi (GSAP + Lottie Powerhouse — RaptorFlow Edition)

This is the dedicated motion skill. Every time the user asks for hover, transition, reveal, loading, daily wins calendar, lock button, proposal apply, or ANY motion — activate this skill immediately. Always redesign the entire component/page from scratch using the full RaptorFlow skeleton. Delete old code completely. Use GSAP for 95% of motion and custom Lottie only for the rare origami bird mascot. Enforce the exact non-negotiable rules from raptorflow-design-vibe at every step. Invoke MCP servers explicitly.

## Locked Motion Philosophy (Repeat in Every Output)
- Quiet power only: subtle, fast, functional
- Motion purposes (only these three):
  - Confirm action
  - Show causality
  - Guide focus
- NEVER: bokeh, sparkles, glitter, wobble, heavy shadows, neon, gradients in motion
- Mascot (origami bird): Extremely rare — only as small loading spinner, empty state emblem, or lock confirmation (tiny, 24px max)
- Respect @media (prefers-reduced-motion: reduce)
- All animations must feel premium, calm, and reversible (Control-first)

## GSAP Usage Rules (Extensive — Mandatory)
- Always import: import { gsap } from "gsap"; import { ScrollTrigger } from "gsap/ScrollTrigger";
- Register plugins in useEffect
- Use timelines for every complex sequence (e.g., daily wins staggered reveal)
- Hover states: scale 1.01 + slight charcoal lighten (no glow)
- Lock button: smooth scale + bird Lottie on success
- Daily wins calendar: GSAP staggerChildren on scroll or on data load
- Right drawer open: slide from right with opacity
- Status change: fade + micro scale
- All durations: 0.2s–0.4s max (fast & snappy)

## Custom Lottie Rules
- Only for origami bird
- Export as JSON from After Effects or LottieFiles
- Use react-lottie-player
- Trigger: loading → play once, empty state → loop gently, lock → success burst (tiny)
- Provide the full JSON snippet in comments when used

## Component-Specific Animation Examples (Always Follow)
Buttons:
- Primary hover: GSAP to slightly lighter charcoal + 1.02 scale
- Lock button: On click → timeline (scale down → bird Lottie → scale up + success color flash in monochrome)

Tables / Calendar (Daily Wins):
- Row enter: stagger fade + slide up
