ImportError: cannot import name 'get_redis' from 'redis.client'

## Root Cause Analysis

The `redis` library version mismatch. The `get_redis` function was previously available in `redis.client` but has been moved or renamed in newer versions of the `redis` library.

## Affected Files

1. `backend/llm.py` - Line 717: `from redis.client import get_redis`

## Solution

Update the import in `backend/llm.py` to use the correct redis client import:

```python
# Old (broken):
from redis.client import get_redis

# New (correct):
from backend.redis_client import get_redis
```

Or use the redis library's native async client:

```python
import redis.asyncio as redis
```
