# RaptorFlow

Strategic Marketing Intelligence Platform

## Fresh Start

This is a clean slate setup with only the landing page.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to see the landing page.

## What's Included

- Landing page (`src/pages/Landing.tsx`)
- Essential configuration files
- Tailwind CSS styling
- Framer Motion animations

## What's Archived

All previous documentation and code has been moved to the `old_docs/` folder for reference.

## Environment

Make sure you have your `.env` file configured with necessary credentials.
